# Heart Disease Classification Using Machine Learning

## Project Overview

This project was completed as part of a Machine Learning assignment. The objective is to build and compare machine learning classification models that predict whether a patient has heart disease based on clinical information.

The project includes:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Random Forest Classifier
- Model Evaluation
- Confusion Matrices
- ROC Curve
- Feature Importance Analysis
- Deployment Strategy
- Model Monitoring

## Dataset

The Heart Disease dataset contains patient information such as:

- Age
- Sex
- Chest pain type
- Resting blood pressure
- Cholesterol
- Fasting blood sugar
- Resting ECG
- Maximum heart rate
- Exercise-induced angina
- ST depression (Oldpeak)
- Number of major vessels
- Thalassemia

**Target Variable:**
- **0** = No Heart Disease
- **1** = Heart Disease

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC

## Results

The Random Forest classifier achieved the best performance with:

- Accuracy: **86.89%**
- Precision: **90.00%**
- Recall: **84.38%**
- F1-Score: **87.10%**
